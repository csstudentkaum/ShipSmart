1. Page Hierarchy

The website follows a flat architecture where all pages are accessible from the main navigation bar within one click from the Home page.

Page File Description
Home index.html Landing page with hero section, tracking form, feature cards, how-it-works steps, and testimonial.
Services pages/services.html Displays four core services: Shipment Tracking, Delivery Scheduling, Real-Time Notifications, and Multi-Carrier Integration.
Schedule pages/schedule.html Weekly shipping schedule table with pickup and delivery windows, important notes, and express delivery zones.
Video pages/video.html Embedded platform demo video with a description of what ShipSmart does.
Feedback pages/feedback.html User feedback form with client-side validation and a success confirmation message.
About pages/about.html Information about the ShipSmart team and project.

File Description

5. Navigation Flow

All pages share the same navigation bar with five links: Home → Services → Schedule → Video → Feedback. The user can reach any page from any other page in a single click (flat architecture).

Every page also includes a consistent footer with page links, social media icons, and copyright information.

Note: All pages share the same header navigation and footer. Print CSS only applies to the Schedule page. The About page is accessible from the footer on select pages.
